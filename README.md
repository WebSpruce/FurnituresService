# ASP .NET MVC - FurnituresService

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## Features

- You are able to register and login a new account.
- In Administration Panel you are able to add, remove, edit and see Users, Orders, Furnitures and Categories.
- In the catalog you are able to see products order by categories.
- You can add products to the cart and order.
